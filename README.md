# PDF Annotation Tool

A lightweight web-based PDF and image annotation tool built using **HTML, CSS, and JavaScript**.
This tool allows users to open documents and add annotations such as shapes, arrows, text, and comments directly in the browser.

## Features

* Open PDF or image files
* Draw shapes (rectangle, circle, arrow, line)
* Add text annotations
* Comment panel for notes
* Export annotations
* Lightweight and browser-based

## Technologies Used

* HTML
* CSS
* JavaScript

## Getting Started

1. Clone the repository

```
git clone https://github.com/yourusername/pdf-annotation-tool.git
```

2. Open the project folder.

3. Run the project by opening:
